A Keras-native library for medical image analysis — backend-agnostic, GPU/TPU-ready, and built for volumetric data.

TensorFlow PyTorch JAX 2D + 3D Multi-GPU / TPU Sliding window

Core Strengths

↔

Backend agnostic

Runs on TensorFlow, PyTorch, or JAX through one environment variable.

API

High-level API

Provides a consistent interface for transforms and model creation.

⇧

Scalable execution

Supports single GPU, multi-GPU, and TPU-VM training workflows.

3D

3D sliding window

Enables efficient patch-based inference for volumetric data.

∩

Built-in metrics

Includes Dice, IoU, and other medical segmentation metrics.
