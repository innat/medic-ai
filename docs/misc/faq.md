# FAQ

## Which Keras backend should I use?

Use the backend that matches your training and deployment environment. `medicai` is designed for Keras 3 workflows across TensorFlow, PyTorch, and JAX backends.

## Does medicai support 2D and 3D medical images?

Yes. Model builders and transformations are designed to support both 2D and 3D medical imaging workflows where the underlying component supports that dimensionality.

## Where are model variants documented?

Model variants are listed under `API Reference > Models`, grouped by Classification and Segmentation.

## Where should examples live?

Examples are organized under the top-level `Code Example` tab.

## How do I report an issue?

Open a GitHub issue at [innat/medic-ai](https://github.com/innat/medic-ai/issues/new/choose) with a minimal reproduction, environment details, and the expected behavior.
